# Memory and CPU Management

Monitoring system resources helps maintain performance.

---

# Commands

| Command | Description |
|------|-------------|
| top | real-time CPU usage |
| free | memory usage |
| vmstat | system statistics |
| htop | interactive process viewer |

Example

```
free -h
```

---

# Lab Exercise

Check memory usage

```
free -h
```

Monitor CPU

```
top
```
