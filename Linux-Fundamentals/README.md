# Linux Fundamentals

## What is Linux?

Linux is a **free and open-source operating system kernel** created by Linus Torvalds.

It manages system hardware and allows users to interact with the system.

Linux is widely used in:

- Cloud computing
- DevOps infrastructure
- Servers
- Containers (Docker, Kubernetes)

---

# Linux Features

- Open source
- Multi-user
- Multitasking
- Secure permission system
- High stability

---

# Linux Architecture

Linux architecture includes:

1. Hardware
2. Kernel
3. Shell
4. Applications

---

# Important Commands

| Command | Description |
|------|-------------|
| uname | Show system information |
| hostname | Show system name |
| whoami | Show current user |
| uptime | Show system uptime |

Example:

```
uname -a
```
