# File and Directory Management

Linux provides many commands to manage files and directories.

---

# Common Commands

| Command | Description |
|------|-------------|
| ls | list files |
| cd | change directory |
| mkdir | create directory |
| rm | delete file |
| cp | copy file |
| mv | move or rename |

---

# Examples

Copy file

```
cp file1 file2
```

Move file

```
mv file1 folder/
```

Delete file

```
rm file1
```

---

# Lab Exercise

Create directory

```

Create file

```

```

Move file

```
