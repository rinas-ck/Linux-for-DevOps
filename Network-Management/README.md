# Network Management

Linux provides many tools to manage networking.

---

# Commands

| Command | Description |
|------|-------------|
| ip a | show IP address |
| ping | test connectivity |
| netstat | network connections |
| curl | fetch data from web |

Example

```
ip a
```

---

# Lab Exercise

Check IP address

```
ip a
```

Test connectivity

```
ping google.com
```
