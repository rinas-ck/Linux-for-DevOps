# 🐧 Linux for DevOps Engineers

A structured Linux learning repository covering **Linux fundamentals, system administration, and DevOps-related Linux skills**.

This repository is designed for **DevOps engineers, cloud engineers, and system administrators** who want to build a strong Linux foundation.

---

# 📚 Topics Covered

Each folder contains detailed explanations, commands, and lab exercises.

| Topic | Description |
|------|-------------|
| Linux Fundamentals | Linux architecture, kernel, distributions |
| File System and Hierarchy | Linux directory structure |
| User and Group Management | Creating and managing users |
| Permission Management | chmod, chown, security |
| File and Directory Management | Basic file operations |
| Hardlink and Softlink | Linux file linking |
| Process Management | Managing running processes |
| Memory and CPU Management | System resource monitoring |
| Network Management | Networking tools and troubleshooting |
| Shell Scripting | Bash scripting basics |
| Cronjob | Automating tasks |
| Monitoring and Logs | Logs, metrics, system monitoring |

---

# 🛠 How to Use This Repository

1️⃣ Navigate through the folders in order  
2️⃣ Read the explanations  
3️⃣ Run commands on a Linux machine / VM  
4️⃣ Practice the lab exercises  

---

# 🎯 Goal

By completing all topics in this repository you will:

✔ Understand Linux deeply  
✔ Be comfortable using the Linux terminal  
✔ Gain skills required for **DevOps / Cloud roles**
